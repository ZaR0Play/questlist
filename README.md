# questlist
#  Консольный менеджер задач (C#)

Простая консольная программа для управления списком задач с возможностью отметки выполнения.

##  Описание

Программа позволяет:
-  Добавлять задачи
-  Просматривать список всех задач
-  Удалять задачи
-  Отмечать задачи как выполненные (True/False)
-  Просматривать статистику выполнения
-  Фильтровать задачи по статусу
-  Сохранять и загружать задачи из файла

##  Запуск

### Требования
- .NET SDK 6.0 или выше
- Visual Studio Code / Visual Studio / любой текстовый редактор

### Способ 1: Запуск через терминал

```bash
# Компиляция
dotnet build

# Запуск
dotnet run
